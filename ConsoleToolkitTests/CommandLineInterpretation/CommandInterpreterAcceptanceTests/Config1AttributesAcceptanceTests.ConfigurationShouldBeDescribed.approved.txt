Available commands                                          
                                                            
c1  Command 1 a file.                                       
c2  Command 2 an archive                                    
c3  Generate loads of spam                                  
                                                            
Description of c1 command:                                  
Command 1 a file.                                           
                                                            
c1 <filename> [options]                                     
                                                            
Parameters:                                                 
                                                            
filename  The name of the file.                             
                                                            
Options:                                                    
                                                            
--delete, -D   Delete the file after                        
               processing.                                  
--archive, -A  Archive after processing                     
                                                            
Description of c2 command:                                  
Command 2 an archive                                        
                                                            
c2 <name> <keep> [options]                                  
                                                            
Parameters:                                                 
                                                            
name  The name of the archive.                              
keep  The number of days to keep the                        
      archive                                               
                                                            
Options:                                                    
                                                            
--maxSize, -M  The maximum size of the                      
               archive.                                     
                                                            
Description of c3 command:                                  
Generate loads of spam                                      
                                                            
c3 <iterations> <message>                                   
[<overrunlength>] [options]                                 
                                                            
Parameters:                                                 
                                                            
iterations     Number of times to repeat                    
message        The message to spam.                         
overrunlength  Amount packet should be                      
               longer than it claims                        
               Default value: 5                             
                                                            
Options:                                                    
                                                            
--kidding, -K  Run in just kidding mode.
