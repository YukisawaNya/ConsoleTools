Available commands                                                                                                                   
                                                                                                                                     
client  Client operations                                                                                                            
                                                                                                                                     
Description of client add command:                                                                                                   
Add a new client with default settings.                                                                                              
                                                                                                                                     
client add <name>                                                                                                                    
                                                                                                                                     
Parameters:                                                                                                                          
                                                                                                                                     
name  The name of the client                                                                                                         
                                                                                                                                     
Description of client del command:                                                                                                   
Delete a client from the system.                                                                                                     
                                                                                                                                     
client del <clientguid>                                                                                                              
                                                                                                                                     
Parameters:                                                                                                                          
                                                                                                                                     
clientguid  The guid of the client
